# 墨祟：走阴录 — 美术方向文档 v3.2

## 核心关键词

水墨 / 暗雅 / 纸质 / 冷寂 / 红点缀

游戏视觉锚定在"墨滴落在宣纸上"的意象。不是明亮的水彩，而是深夜书房里一支毛笔蘸墨的感觉——大量暗色留白、粗糙纸纹、偶尔一抹朱红。

## 色彩体系

### 主调色板（gamedata.js `C` 对象 + styles.css CSS 变量）

| 用途 | 色值 | 变量名 | 说明 |
|------|------|--------|------|
| 纸底 | `#f1e6d4` | `--paper` / `C.paper` | 所有背景基底色，温暖米黄 |
| 纸边 | `#d9c9af` | `--paper-deep` / `C.edge` | 卡片边框、分隔线 |
| 墨色 | `#171310` | `--ink` / `C.ink` | 主要文字、敌人轮廓、UI线条 |
| 淡墨 | `#2c2520` | `--ink-soft` / `C.soft` | 次要文字、暗部层次 |
| 朱砂 | `#a33a2d` | `--accent` / `C.accent` | 唯一的亮色强调——暴击、Boss名、危险 |
| 灰褐 | `#8d7d69` | `--ash` / `C.ash` | 提示文字、环境粒子 |
| 苔绿 | `#4d6156` | `--moss` / `C.moss` | 治疗、自然元素 |
| 幽白 | `rgba(220,210,190,0.7)` | `C.ghost` | 鬼魂/幽灵半透明 |
| 金铜 | `#9c835a` | `--gold` / `C.gold` | 成就、Boss击杀、稀有掉落 |

### 色彩使用原则

1. **90% 非饱和色** — 画面由 paper/ink/soft/ash 组成，保持暗雅基调
2. **accent 只用于"该注意的东西"** — 暴击数字、Boss名、闪避按钮激活态、危险标记
3. **gold 只用于"特殊时刻"** — Boss击杀、成就解锁、稀有遗物
4. **moss 保持克制** — 治疗效果、个别自然类遗物
5. **禁止引入新色相** — 除非是新增敌人类型的专属色彩，且需在 gamedata.js `C` 中注册

## 线条与形状

### 敌人

- **轮廓线**：2-3px，ink色，不规则抖动（`drawBlob` 函数）
- **填充**：低透明度（0.3-0.5），配合类型专属色
- **大小**：圆形碰撞体，r=10-30，不追求写实
- **原则**：墨点感，不是卡通角色

### 玩家

- 轮廓 2.5px ink色
- 填充 paper 色（与背景区分靠轮廓线）
- 武器用类型色画简笔

### 攻击特效

- **近战弧**：半透明扇形，accent 色
- **弹道**：圆点+短拖尾
- **AOE**：扩散圆环，高透明度
- 所有特效 lifetime 30-60 帧，不持续

## 字体

| 用途 | 字体栈 | CSS 变量 |
|------|--------|----------|
| 标题/印章 | STKaiti, KaiTi, Kaiti SC, Source Han Serif SC, serif | `--title-face` |
| 正文/UI | system-ui, sans-serif | 默认 |
| 数字/调试 | monospace | — |

原则：所有中文标题用楷体（Kaiti），不用黑体。数字和英文用系统字体。

## Canvas 绘制规范

### 纸纹背景

- `buildBg()` 生成离屏 canvas，只画一次
- 基底 paper 色 + 随机深浅噪点 + 边缘暗角
- 不加明显纹理图案（保持简洁）

### 阴影使用

- `setShadow(c, col, blur, g)` — 只在 `_pm >= 0.7` 时启用
- shadowBlur 不超过 12px
- 颜色用 ink 低透明度，不用纯黑

### 粒子

- 尺寸 1.5-6px，生命周期 20-60 帧
- 类型：ink / accent / moss / gold / ghost / soul
- 用 `spawnP()` 生成，不手动画圆

## UI 风格

### 卡片（武器/遗物/进化）

- 纸色背景 + ink 边框
- 左上角彩色方块表示类型
- 文字用 ink/soft 色
- 选中态：accent 边框 + 轻微放大

### HUD

- 顶部半透明条，不遮挡战斗
- 血条用 ink→accent 渐变
- 波次/斩杀用 soft 色

### 按钮

- 圆角（8-16px）
- paper 背景 + ink 边框
- hover/active: accent 渐入
