# assets/sprites — 角色和物体精灵

存放敌人、玩家、武器等游戏内精灵图。

## 命名规则

- `sprite-{类型}-{方向?}.{png}` — 例：`sprite-zhikui.png`
- 尺寸：根据实体碰撞半径，@2x 制作
  - 小敌人(r=12)：48x48px
  - 中敌人(r=18)：72x72px
  - 大敌人/Boss(r=25+)：100x100px
  - 玩家：64x64px
- 格式：透明背景 PNG

## 注意

当前游戏所有角色用程序绘制（drawBlob 圆形）。精灵图是未来增强方向，不阻塞当前版本。

接入方式：在 render() 中检查 `sprite` 属性，有则 drawImage，无则走现有程序绘制。
