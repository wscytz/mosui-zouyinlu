# 墨祟：走阴录 — 第一批资产清单

> 设计参考 / 历史清单：本文记录第一批资产规划，不代表 v13.0 的完整资产缺口。后续 v13.2 图标管线应先生成缺图清单和优先级，再更新本文或新建 manifest。

> 优先级参考 ART_DIRECTION.md。标注"可直接进游戏"的资产做好后即可接入，无需额外代码改动。

## 第一优先 — 直接提升观感

### A0. 主封面 / 标题页主视觉

| 属性 | 值 |
|------|-----|
| 用途 | 标题页背景 + 宣传图 + README 头图 |
| 尺寸 | 1536x2048px（竖版） |
| 格式 | 透明要求不高，可 PNG 或 JPG |
| 风格 | 孤身夜行客 / 牌楼 / 巨大墨影 / 朱砂红月，留出标题安全空间 |
| 接入 | 标题页主视觉，后续可作为游戏宣传封面 |
| 可直接进游戏 | 是 |
